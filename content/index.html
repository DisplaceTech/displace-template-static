About This Site

This is a template static site created with the Displace CLI. It demonstrates a modern, scalable deployment architecture using:

  • 🐳 Docker - Multi-stage builds with Python/uv and Caddy
  • ☸️ Kubernetes - Production-ready manifests
  • ⚡ Caddy - High-performance web server with automatic HTTPS
  • 🔄 Content Sync - Update content without rebuilding containers

Key Features

🚀 Fast Deployment

Deploy your static site to Kubernetes with a single command

🔄 Content Sync

Update content without rebuilding containers using PVC synchronization

🔒 Security First

Automatic HTTPS, security headers, and non-root containers

📈 Scalable

Built for production with resource limits and health checks

Getting Started

Ready to customize this template? Here's how to get started:

  1. Edit the content in the content/ directory
  2. Run make sync-content to update the live site
  3. For major changes, use make build and make deploy
  4. Visit the Displace CLI documentation for more advanced features
